# 此刻在场 — 网页内容概览

> 腾讯视频 AI 广告体验编排 Demo · 单页 HTML · 零依赖
> 品牌: MOMERA 此刻 · 出品人: SELH · 2026

## 页面结构（8 个区块，自上而下）

### 00 · Hero 首屏
- 主标题「此刻在场」
- BY SELH 署名
- 副标题: 让广告从「卖曝光」升级为「卖场景、卖参与、卖表达」
- 传统广告 vs AI 广告 左右对比卡片（红/蓝）
- CTA: 开始体验 DEMO / 查看能力矩阵
- 顶部固定导航栏，7 个锚点链接

### 01 · FUNC 01 广告版本路由矩阵
- 4 张卡片展示广告版本路由逻辑
- 根据用户状态（新用户/老用户/高活跃/低活跃）分发不同广告素材

### 02 · FUNC 02 情绪窗口判断
- 交互式视频 Demo
- 两个模式切换按钮: 传统广告介入 / AI 缓冲介入
- 传统模式: 高潮片段 → 暂停 → 插入广告 → 恢复正片
- AI 模式: 缓冲片段 → 暂停 → 插广告 → 恢复缓冲
- 展示 AI 选择「合适的时机」而非「所有时机」投广告

### 03 · FUNC 03 情绪拍立得
- 多步骤交互流程:
  1. 选择情绪关键词（想留住 / 有点累 / 陪我一下）
  2. 拍照或上传照片
  3. AI 生成带情绪文案的拍立得卡片
  4. 可保存、分享、重拍
- 数字人形象区 + 示例场景

### 04 · FUNC 04 中场竞猜
- NBA 场景模拟（湖人 vs 勇士 · 中场休息）
- 阵营选择互动（选湖人/选勇士）
- AI 在中场缓冲窗口插入参与型广告

### 05 · FUNC 05 AI 场景改造展示
- 同一产品/人物，AI 根据用户场景重构广告背景
- 三张对比图（原始 → 改造后 × 3 个场景）
- 60/40 图文布局

### 06 · 数字资产库
- 视频素材展示: 正片高潮片段、缓冲片段、数字人视频、广告片
- 图片素材: Logo、Slogan 卡、数字人肖像、拍立得示例、场景改造示例
- 代码风格标签展示（DM Mono 字体 + 黑色卡片）

### 07 · CONCLUSION 商业价值
- 四列数据卡片（GMV 提升 / 时长贡献 / 转化率 / CPM 溢价）
- 底部 Footer: MOMERA 品牌信息 + SELH 署名

## 交互功能
- 7 段视频播放逻辑（传统/AI 两种模式）
- 情绪关键词选择 → 拍立得生成流程
- 竞猜阵营投票
- IntersectionObserver 导航高亮跟踪
- 多处倒计时/进度条动画

## 技术栈
- 纯 HTML + CSS + JS
- 字体: DM Mono + Noto Sans SC（Google Fonts）
- 配色: 蓝 #0B38C8 / 红 #D92B1E / 黄 #FFE100 / 墨 #111 / 纸 #F2F2EF
- Vercel 部署，vercel.json 全量 rewrite 到 demo.html
